数据质量分析

目的是为了数据预处理做准备，为数据挖掘分析结论有效性和准确性提供基础。

  1. 脏数据:不符合要求，以及不能直接进行相应分析的数据。包括：
    1. 缺失值
    2. 异常值
    3. 不一致的值
    4. 重复数据及含有特殊符号的数据

缺失值分析

指某个记录的缺失和记录中某个字段信息的缺失。

产生原因

  1. 某些信息暂时无法获取
  2. 有些信息被遗漏
  3. 属性值不存在

影响

  1. 数据挖掘建模将丢失大量有效信息
  2. 挖掘的不确定性更加显著
  3. 包含空值的数据会使建模过程陷入混乱，导致不可靠的输出。

缺失值分析

  1. 使用统计分析可以得到缺失值的属性的个数，以及每个属性的未缺失数，缺失数和缺失率。

缺失值处理

  1. 对可能值进行插补和不处理等，见第四章

异常值分析

  1. 异常值指样本中的个别值，其数值明显偏离其余的观测值。
  2. 异常值分析：用于检验数据是否有录入错误以及不合常理的数据。

简单统计量分析

先对变量做一个描述性统计，进而查看哪些数据是不合理的。如给数据设置最大值和最小值。

3σ原则

如果数据符合正态分布，在3σ原则下，异常值被定义为一组测定值中与平均值的偏差超过3倍标准差的值。(即使不服从正态分布，也可以用原理平均值的多少倍标准差来描述)

箱型图分析

  1. 异常值通常被定义为小于QL-1.5IQR或大于QU+1.5IQR。

  2. QL称为下四分位数，表示全部观察值中只有四分之一的数据取值比它小

  3. QU称为上四分位数，表示全部观察值中有四分之一的数据取值比他大

  4. IQR 为四分位数间距。IQR=QU-QL，其间包含了全部观察值的一半。

  5. 优点：

    1. 不对数据的分布形式有要求
    2. 判断异常值的标准时以四分位数和四分位距为标准，具有一定的鲁棒性。(25%以上的异常值才可能对四分位数产生较大的扰动)
    3. 箱型图识别异常值的结果比较客观。

  6. 使用

  7. 异常值检测

一致性分析

数据不一致

是指数据的矛盾性。

原因

  1. 不一致性的产生主要发生在数据集成的过程中，可能是因为数据来自于不同的数据源
  2. 对重复存放的数据未能进行一致性更新造成的。(两张表中的同一个数据只改了一个)

数据特征分析

分布分析

目的是揭示数据的分布特征和分布类型

定量数据的分布分析

  1. 想要了解分布形式是否对称，是否存在某些特大或特小的可疑值。可绘制频率表格，绘制频率直方图，绘制茎叶图来分析

  2. 主要问题： 选择组数和组宽

  3. 步骤

    1. 求极差
    2. 决定组数和组距
    3. 决定分点
    4. 列出频率分布表
    5. 绘制频率分布直方图(纵轴频率密度)

  4. 原则

    1. 各组间必须相互排斥
    2. 各组必须将所有的数据包含在内
    3. 各组的组宽最好相等

定性数据的分布分析

  1. 根据变量的分类类型来分组，使用饼图和条形图来描述分布。
  2. 条形图的高度和饼图的面积用来代表每一类型的百分比或频数。

对比分析

  1. 对比分析：指把两个相互联系的指标进行比较，从数量上展示和说明研究对象规模的大小，水平的高低，速度的快慢，以及各种关系是否协调。
  2. 关键在于选择对比标准
  3. 主要包括绝对数比较，相对数比较
  4. 相对数标胶：通过两个有联系的指标对比计算，用以反应客观现象间数量的联系程度的综合指标。相对数包括
    1. 结构相对数：同一总体内的部分数值与全部数值的比重。
    2. 比例相对数：同一总体中，不同部分的数值的对比，表明总体中各部分的比例关系
    3. 比较相对数：将同一时期的两个性质相同的指标数值进行对比，说明同类现象在不同的空间条件下的数量对比关系。
    4. 强度相对数：将两个性质不同但有一定联系的总量指标进行对比，用以说明现象的强度、密度和普通程度。
    5. 计划完成程度相对数：对某一时期实际完成数与计划数对比，用以说明计划完成程度。
    6. 动态对比数：同一现象在不同时期的指标数值进行对比，用以说明发展方向和变化的速度。

统计量分析

  1. 集中趋势度量

    1. 均值(加权均值)
    2. 频率分布表的加权平均数
    3. 中位数：先排序
    4. 众数

  2. 离中趋势度量

    1. 极差:对极端值非常敏感

    2. 标准差

    3. 变异稀疏：度量标准差相对于均值的离中趋势

    4. 四分位数间距：需要排序。间距越大说明数据的变异程度越大。

周期性分析

  1. 周期性分析:探索某个变量是否随着时间变化呈现出某种周期变化趋势。

贡献度分析(帕累托分析)

相关性分析

  1. 用于分析连续变量之间线性相关程度的强弱，并用适当的统计指标表示。表示方法包括

  2. 直接绘制散点图

  3. 绘制散点图矩阵

  4. 计算相关系数

    1. Pearson相关系数：要求连续变量的取值服从正态分布

    2. Spearman秩相关系数

      1. 只要两个变量具有严格单调函数关系，那么他们就是完全Sperman相关的。

    3. 判断系数：相关系数的平方；用以衡量回归方程对y的解释程度。

基本统计特征函数

  1. D.corr(method=‘pearson’)，样本D可为DataFrame，返回相关系数矩阵，method参数为计算方法，支持pearson，kendall，spearman系数

  2. S1.(S2,method=‘pearman’)，求S1和S2之间的相关系数。

  3. cov()返回协方差矩阵

    1. 样本D可为DataFrame
    2. 可用S1.cov(S2)计算两个Series之间的协方差

  4. skew/kurt

    1. 计算数据样本的偏度(三阶矩)/峰度(四阶矩)

  5. describe

    1. 返回一些基本统计量

拓展统计特征函数

  1. DataFrame或Series的对象D；D.cumsum()
  2. pd.rolling_mean(D,k)每k列计算一次均值，滚动计算。

数据挖掘基本概念

  1. 数据挖掘：从大量数据中挖掘处隐含的，未知的，对决策有潜在价值的关系，模式和趋势，并用这些知识和规则建立用于决策支持的模型。
  2. 基本任务：包括利用分类与预测，聚类分析，关联规则，时序模式，偏差检测，智能推荐等方法，提取数据中的价值。

数据挖掘建模过程

定义挖掘目标

  1. 即明确系统完成后达到什么样的效果。
  2. 需要分析应用领域，包括应用中的各种知识和应用目标。

数据取样

  1. 抽取标准：相关性；可靠性；有效性。
  2. 一定要保证数据的质量，衡量数据质量的标准
    1. 资料完整无缺，各类指标项齐全
    2. 数据准确无误，反应的都是正常状态下的水平

常见抽样方式

  1. 随机抽样：使数据集中的每一组观测值都有相同的被抽样的概率
  2. 等距抽样
  3. 分层抽样：先将数据分成若干层次，在每个层次中的观测值都具有相同的被选用的概率，但对不同层次可设定不同概率
  4. 从起始顺序抽样
  5. 分类抽样：依据某种属性的具体取值来选择数据子集，如按客户名称分类。

数据搜素

目的是对所抽取的样本数据进行搜素，审核和必要的加工处理，以保证最终的挖掘模型的质量所必须的。

数据预处理

  1. 目的是减少噪声和不一致，不完整的数据。
  2. 主要包括：数据筛选；数据变量转换，缺失值处理，坏数据处理，数据标准化，主成分分析，属性选择，数据规约等

挖掘建模

  1. 首先分析本次建模属于哪类问题（分类与预测，聚类，关联规则，时序模式或者智能推荐）

有趣的数

问题描述

我们把一个数称为有趣的，当且仅当：
　　1. 它的数字只包含0, 1, 2, 3，且这四个数字都出现过至少一次。
　　2. 所有的0都出现在所有的1之前，而所有的2都出现在所有的3之前。
　　3. 最高位数字不为0。
　　因此，符合我们定义的最小的有趣的数是2013。除此以外，4位的有趣的数还有两个：2031和2301。
　　请计算恰好有n位的有趣的数的个数。由于答案可能非常大，只需要输出答案除以1000000007的余数。

输入格式

输入只有一行，包括恰好一个正整数n (4 ≤ n ≤ 1000)。

输出格式

输出只有一行，包括恰好n 位的整数中有趣的数的个数除以1000000007的余数。

样例输入

4

样例输出

3

思路

  1. 由于规则限制，每一位能选的数都与之前位的状态有关，所以是个dp问题

  2. 将状态进行分类可得，前i位数字使用情况包括

    1. 只含2
    2. 只含2、0
    3. 只含2、3
    4. 只含2、0、3
    5. 只含2、0、1
    6. 含4种数字。

  3. 如果现在要尝试填充第i位，第i位则有6中状态。以状态5为例,设j = i - 1。则第i位的来源包括

    1. 第j位的状态5扩展，在第i位填1或2
    2. 从j位的专题2扩展，在第i位填1
    3. dp[i][4] = (dp[i - 1][1] + dp[i - 1][4] * 2) % Mod;

城市规划

问题描述：

思路

  1. 有这样一条边，该边将树分割成两部分，其上下方子树的结点间路径需要通过该边。设该边权重为w，上方子树中有重要结点t个，下方子树有重要结点k-t个，则该边对结果的贡献为t*(k-t)*w，其中t*(k-t)表示上下方子树中结点的一种组合。
  2. 现用d[u][i]来表示以u结点为根的树中选择i个重要结点的情况下的最短路径和。
  3. d[u][i+j]的状态可能来源于d[u][i]+d[v][j]，其中v是u的子树。可理解为从u的树中选取i个重要节点，从其子树中选择j个结点。则d[u][i + j] = min(1ll*d[u][i + j], 1ll* buf[u][i] + d[v][j] + j * (k - j)*w); 注意此处的1ll转型非常重要，虽然最终结果一定在int内，但是中间的路径和可能超过。包含路径和的问题都应该作此考虑
